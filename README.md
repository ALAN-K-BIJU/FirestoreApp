# 🔥 FirestoreApp

**FirestoreApp** is an Android application developed in Java that demonstrates how to perform basic CRUD operations using **Firebase Firestore**. It’s a practical app designed to help you understand how to interact with Firestore documents, collections, and real-time updates in a clean and simple UI.

---

## 🎯 Features

- 📝 Add data to Firestore (create new documents)  
- 📖 Read and display Firestore data in real-time  
- ✏️ Update existing Firestore documents  
- 🗃️ Structured collection/document usage  
- ⚡ Smooth and responsive user experience
